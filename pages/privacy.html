Overview

This Privacy Policy explains how the Budget Dashboard application (“we”, “us”) collects, uses, and stores data. This is currently a personal project/limited rollout. Replace bracketed placeholders before broader use.

Data we collect

  • Account data: email address and profile information used for authentication.
  • Budgeting data: budgets, categories, members, and transactions you create in the app.
  • Imported financial data: if you connect a bank via Plaid Link, Plaid may provide account and transaction data used to populate your budgeting dashboard.
  • Operational data: basic logs from hosting/providers for debugging and security.

How we use data

  • Provide budgeting features (dashboards, charts, reports).
  • Import and categorize transactions you request from connected accounts.
  • Maintain and secure the service (fraud prevention, troubleshooting).

Third-party services

  • Plaid: used to connect financial accounts and retrieve data you authorize via Plaid Link.
  • Supabase: used for authentication and data storage (if enabled).
  • Hosting provider: used to run the application and deliver the site over HTTPS.

Sharing

We do not sell personal data. We share data only with service providers needed to operate the application (e.g., Plaid, Supabase, hosting), or if required by law.
